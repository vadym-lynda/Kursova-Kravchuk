Проект реалізований на LocalStorage (тобто працює у бразері без використання будь-якого сервера)

Для демонстрації проекту відкрити файл index.html

Для початку демонстрації натиснути зелену кнопку "Prepare".
Вона очистить LocalStorage для сторінки і додасть туди параметри нових 5-ти елементів (чашок)

Для створення елемента необхідно натиснути на посилання "Create cup" (ліва сторона хідера сайту)
При створенні елементу потрібно, окрім введення його параметрів, також вибрати для нього зображення.
!!! Зображення слід вибирати ТІЛЬКИ з папки [img]. В цю папку може бути поміщено зображення будь-яких розмірів.
В кінцевому варіанті CSS приводить їх до потрібного розміру. Рекомендовано поміщати в цю папку зображення
з співідношенням сторін 1:1 (тобто квадратні). Тоді вони не будуть спотворені CSS при виводі.

Пошук є НЕчутливим до регістру (case insensitive)

Сортування реалізоване по зростанню (можна заміною одного знака в коді зробити його "по спаданню")
